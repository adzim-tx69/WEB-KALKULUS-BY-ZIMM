Materi ringkas: Turunan, Limit, dan Integral
Turunan — definisi, aturan, contoh

Turunan mengukur laju perubahan fungsi: f'(x)=lim_{h→0} (f(x+h)-f(x))/h.

Aturan dasar

  • Const: (c)' = 0
  • Pangkat: (x^n)' = n x^(n-1)
  • Produk: (uv)' = u'v + uv'
  • Rantai: (f(g(x)))' = f'(g(x))·g'(x)
  • Trigonometri: (sin x)' = cos x; (cos x)' = -sin x

Contoh

  • f(x) = x^3 - 2x
  • f(x) = sin(x^2)
  • f(x) = x*exp(x)
Limit — definisi, teknik, contoh

Limit adalah perilaku fungsi saat x mendekati nilai tertentu. Teknik: substitusi, faktorisasi, L'Hôpital.

Contoh

  • lim x→0 (sin x)/x = 1
  • lim x→0 (sqrt(x+1)-1)/x = 1/2
Integral — definisi, aturan, contoh

Integral mengukur akumulasi/luas: ∫_a^b f(x) dx = F(b)-F(a) jika F' = f.

Contoh

  • ∫_0^1 x^2 dx = 1/3
  • ∫ sin x dx = -cos x
